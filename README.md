# Хмель & Солод

Одностраничный адаптивный сайт частной пивоварни.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```powershell
python -m http.server 8000
```

Затем откройте `http://localhost:8000`.

## Структура

- `index.html` — разметка сайта
- `styles.css` — стили, адаптивность и анимации
- `script.js` — каталог, фильтры, корзина и формы
- `assets/brewery-hero.png` — главный фирменный визуал
- `assets/beer-*.png` — отдельные фотографии для каждого сорта
- `assets/drink-*.png` — фотографии сидра и настоек
- `assets/food-*.png` — фотографии закусок
- `assets/brewery-team.png` — фотография команды в пивоварне
- `api/order.js` — безопасная отправка заказов в Telegram через Vercel

## Возможности магазина

- конструктор собственного набора из шести бутылок со скидкой 10%;
- пиво недели с таймером акции;
- остатки товаров и карты вкуса;
- выбор даты и времени доставки;
- интерактивная карта зоны доставки;
- локальная проверка статуса последнего заказа;
- QR-код для открытия меню на телефоне.

## Telegram-заказы на Vercel

В настройках проекта Vercel добавьте две переменные окружения:

- `TELEGRAM_BOT_TOKEN` — токен, выданный BotFather
- `TELEGRAM_CHAT_ID` — ID чата, куда должны приходить заказы

После добавления переменных выполните Redeploy проекта.
